# Functional E-commerce Orange Accent Design System

## What It Is

This MDC (Markdown Cursor) file contains a comprehensive design system extracted from **Amazon.co.uk**, one of the world's largest e-commerce platforms. This design system represents a functional, conversion-focused e-commerce design with a signature orange accent color.

The design prioritizes usability, product discovery, and clear calls-to-action. It features a clean, organized layout optimized for both desktop and mobile shopping experiences. The design system is perfect for building e-commerce websites, marketplaces, or any product-focused web application.

## Design Style Characteristics

### Visual Characteristics
- **Clean and Functional**: Minimal decorative elements, focus on usability
- **Information-Dense**: High information density with clear visual hierarchy
- **Product-Focused**: Design elements support product discovery and purchase
- **Professional**: Trustworthy and reliable aesthetic

### Color Palette
- **Primary Orange**: `#FF9900` - Used for CTAs, buttons, and brand recognition
- **Neutral Grays**: Clean backgrounds and borders for structure
- **Semantic Colors**: Success green, error red, info blue for clear communication
- **High Contrast**: Excellent readability and accessibility

### Typography
- **Font Family**: Amazon Ember (or similar sans-serif)
- **Clear Hierarchy**: Size and weight variations guide user attention
- **Readable**: Adequate line heights and spacing
- **Responsive**: Font sizes adapt to screen size

### Layout Patterns
- **Grid-Based**: CSS Grid for flexible product layouts
- **Card Components**: Consistent product and deal cards
- **Sticky Navigation**: Always-accessible navigation bar
- **Responsive**: Mobile-first approach with breakpoints

### Animation Style
- **Subtle**: Minimal animations, focus on functionality
- **Smooth**: 0.2s ease transitions for interactions
- **Purposeful**: Animations guide user attention and provide feedback
- **Performance-Optimized**: CSS-based animations for smooth performance

## Key Design Elements

### Components
- **Product Cards**: Clean cards with images, titles, ratings, and prices
- **Buttons**: Primary orange buttons, secondary buttons, link buttons
- **Navigation**: Sticky header with search, account, and cart
- **Forms**: Clean input fields with focus states
- **Badges**: Prime badges, deal badges, bestseller badges
- **Rating System**: Star ratings with review counts
- **Modals**: Smooth modal dialogs for additional information

### Layout Systems
- **Grid System**: Flexible product grids that adapt to screen size
- **Container System**: Max-width containers with responsive padding
- **Breakpoints**: Mobile (320px), Tablet (768px), Desktop (1024px+)

### Interactive Elements
- **Hover States**: Subtle elevation and color changes
- **Focus States**: Clear focus indicators for accessibility
- **Loading States**: Skeleton loaders and spinners
- **Error States**: Clear error messaging and validation

### Color System
- **Primary Orange**: `#FF9900` for CTAs and brand
- **Link Blue**: `#0066C0` for navigation and links
- **Text Grays**: Multiple shades for hierarchy
- **Semantic Colors**: Success, error, warning, info
