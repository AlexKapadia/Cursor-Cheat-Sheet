# Eleven Labs MCP Integration Guide

Complete documentation for integrating Eleven Labs' Agents Platform using the Model Context Protocol (MCP).

## Files

- `eleven-labs-mcp.mdc` - Comprehensive guide covering platform setup, MCP configuration, workflow design, and best practices

## Quick Start

1. Create an Eleven Labs account and retrieve your API key
2. Configure the MCP server in your development environment
3. Follow the guide in `eleven-labs-mcp.mdc` to build your first agentic application
