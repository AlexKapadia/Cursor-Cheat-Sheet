## What It Is

This MDC (Markdown Cursor) document is a comprehensive extraction and organization of the survey paper **"Deep Learning based Recommender System: A Survey and New Perspectives"** by Shuai Zhang, Lina Yao, Aixin Sun, and Yi Tay (2018, ACM Computing Surveys).

This MDC provides:
- **Complete taxonomy** of deep learning based recommendation models
- **Comprehensive summary** of state-of-the-art approaches
- **Key concepts and techniques** used in deep learning for recommendation
- **Implementation patterns** and architectural designs
- **Mathematical foundations** and formulas
- **Best practices** and recommendations
- **Practical applications** and use cases

The MDC is designed to serve as a complete reference for understanding and implementing deep learning based recommender systems, covering everything from basic neural collaborative filtering to advanced architectures like attention mechanisms, graph neural networks, and reinforcement learning approaches.

## Key Concepts

This survey covers numerous key concepts in deep learning based recommendation:

1. **Neural Collaborative Filtering:** Using neural networks to model user-item interactions
2. **Autoencoder-based Recommendation:** Learning efficient representations through autoencoders
3. **CNN-based Recommendation:** Using convolutional networks for structured data
4. **RNN/LSTM for Sequential Recommendation:** Modeling temporal patterns in user behavior
5. **Attention Mechanisms:** Focusing on relevant features for better recommendations
6. **Multi-modal Recommendation:** Integrating text, images, and other data types
7. **Graph Neural Networks:** Modeling relationships as graphs
8. **Reinforcement Learning:** Using RL for sequential decision making in recommendation
9. **Context-Aware Recommendation:** Incorporating temporal, spatial, and social context
10. **Hybrid Architectures:** Combining multiple deep learning approaches
